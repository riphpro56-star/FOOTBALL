KOORA LIVE - NETLIFY API\n\nهذه نسخة تعمل مع Netlify Functions.\n\nأفضل طريقة رفعها:\n1. فك الضغط.\n2. ارفع الملفات إلى GitHub.\n3. في Netlify اختر Add new site > Import from Git.\n4. اختر المشروع.\n5. Build command: اتركه فارغ.\n6. Publish directory: اكتب .\n7. بعد النشر ادخل Site configuration > Environment variables.\n8. أضف:\n   FOOTBALL_DATA_TOKEN = مفتاح football-data.org\n   WATCH_LINK = رابط CPA أو رابط مشاهدة قانونية\n9. اضغط Redeploy.\n\nإذا لم تضف FOOTBALL_DATA_TOKEN سيعرض الموقع بيانات تجريبية.\n
